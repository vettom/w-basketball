Improving self with SMART Goals

Evey player knows that they have to improve, however it is not easy to improve everything at a time. Hence the need for setting SMART goals for individuals, it can be set by Coach or even by yourself.

SMART

  • Specific : Target specific area like wheelchair skills, turning speed, defence positioning etc
  • Measurable : Have means of measuring progress. Like shooting percentage in the key, outside key. Travel speed with/without ball etc.
  • Achievable : Goals must be smaller challenging and achievable.
  • Realistic : Is it realistic to achieve the goals in specified time frame? Does the player has enough time/resource?
  • Timely : It is important to set timeline for the goals so it can be reviewed, assessed and set new goals.
    • To be where you have never been before, you have to do what you have never done before!
